# Anomaly Detection System (Real-Time)

A real-time anomaly detection system for time-series sensor/log data using **Kafka**, **InfluxDB**, and **TensorFlow (LSTM Autoencoder)**.

## Features

- Live data ingestion using Kafka
- Data preprocessing pipeline (scaling + windowing)
- LSTM Autoencoder anomaly detection
- Stores predictions & anomalies in InfluxDB
- Alert system (Email/Webhook)
- Dashboard for monitoring (Streamlit)

## Tech Stack

- Python
- TensorFlow (LSTM Autoencoder)
- Kafka
- InfluxDB
- FastAPI (API layer)
- Streamlit (Dashboard)
